# chat-manager
Чат менеджер для бесед ВКонтакте

### Установка
```
git clone https://github.com/VorDmi/chat-manager
cd chat-manager
npm install
```

### Возможности
* Заглушить пользователя (!mute)
* Забанить пользователя (!ban)
* Кикнуть пользователя (!kick)
* Выдать предупреждение пользователю (!warn)
